# Nibe SMO S40 Card

A Home Assistant Lovelace custom card for the **Nibe SMO S40** heat pump.

Displays:
- Live temperature gauges with automatic severity color (supply, return, outdoor, discharge)
- Pressure tiles (hi/lo/condenser)
- Refrigerant circuit sensor rows
- Energy grid with automatic COP calculation
- Compressor & run time rows

Visual style matches the [Kids-Room-Dashboard-Card](https://github.com/robman2026/Kids-Room-Dashboard-Card) — dark theme, arc gauges, glow effects.

See the [README](README.md) for full configuration options.
